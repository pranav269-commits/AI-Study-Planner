# CFAI Smart Study Planner - Modular Version

This is the modular version of the Streamlit CFAI Smart Study Planner project for KLH University.

## Folder Structure

```text
CFAI_Study_Planner_Modular/
├── app.py
├── requirements.txt
├── README.md
└── modules/
    ├── data.py
    ├── state.py
    ├── algorithms.py
    ├── utils.py
    ├── graph_utils.py
    └── pages.py
```

## Run

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
